プライバシーポリシー / Privacy Policy

1. アプリについて / About this App

本アプリ「PresenFinder」は、ローカル環境で動作するPowerPointファイル閲覧アプリケーションです。

"PresenFinder" is a local PowerPoint viewer application.

2. 個人情報の収集について / Collection of Personal Information

本アプリは、ユーザーの個人情報を一切収集・送信しません。

This application does not collect or transmit any personal information.

3. ネットワーク通信について / Network Communication

本アプリはインターネット通信を行いません。すべての処理は端末内で完結します。

This app does not access the internet. All processing is performed entirely on the local device.

4. 第三者への提供について / Sharing with Third Parties

ユーザー情報を第三者に提供することはありません。

No user data is shared with third parties.
